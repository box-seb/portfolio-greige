

## Colour Scheme Refresh

### Overview
Replace the current lavender ombre + muted teal palette with a clean, solid greige + blue-violet scheme that feels polished and minimalistic.

### Colour Palette

- **Background**: Warm greige (a soft grey-beige, e.g. `hsl(40, 10%, 94%)`) — easy on the eyes, never pure white
- **Cards / Surfaces**: Slightly lighter warm grey (e.g. `hsl(40, 8%, 97%)`) with subtle borders in a medium greige
- **Text**: Deep charcoal (e.g. `hsl(240, 10%, 15%)`) for headings, softer grey for secondary text
- **Accent (Primary)**: Rich blue-violet (`hsl(239, 84%, 67%)` / ~#6366F1) — used for CTA buttons, links, and interactive highlights
- **Accent hover**: Slightly deeper violet for button hover states
- **Borders & dividers**: Light warm grey, no harsh contrast
- **Muted elements**: Medium greige for placeholder text and secondary UI

### What Changes
1. **CSS variables** in `index.css` updated to the new greige + blue-violet palette (both light and dark mode)
2. **No gradient/ombre effects** — all backgrounds are flat, solid colours
3. **Buttons**: Primary CTA gets the blue-violet fill with white text; secondary buttons get a subtle greige border
4. **Overall feel**: Clean, warm, polished — the greige keeps it soft while the blue-violet adds a distinctive pop of colour
